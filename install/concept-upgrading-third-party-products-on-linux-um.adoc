= Linux에서 타사 제품 업그레이드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Linux 시스템에 설치된 Unified Manager에서 JRE, MySQL과 같은 타사 제품을 업그레이드할 수 있습니다.

이러한 타사 제품을 개발하는 회사는 정기적으로 보안 취약점을 보고합니다. 사용자 일정에 따라 이 소프트웨어의 새 버전으로 업그레이드할 수 있습니다.
