= 성능 이벤트 및 경고 이해
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
성능 이벤트는 미리 정의된 조건이 발생하거나 성능 카운터 값이 임계값을 초과할 때 Unified Manager에서 자동으로 생성되는 알림입니다. 이벤트는 모니터링되는 클러스터의 성능 문제를 파악하는 데 도움이 됩니다.

특정 심각도 유형의 성능 이벤트가 발생할 때 자동으로 e-메일 알림을 보내도록 알림을 구성할 수 있습니다.
