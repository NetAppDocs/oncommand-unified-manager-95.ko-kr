= 그래픽 인터페이스 및 탐색 경로
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager는 유연성이 탁월하며 다양한 방법으로 여러 작업을 수행할 수 있습니다. Unified Manager에서 작업하는 경우 다양한 탐색 경로를 찾을 수 있습니다. 가능한 모든 탐색 조합을 표시할 수 있는 것은 아니지만 몇 가지 일반적인 시나리오에 익숙해야 합니다.
