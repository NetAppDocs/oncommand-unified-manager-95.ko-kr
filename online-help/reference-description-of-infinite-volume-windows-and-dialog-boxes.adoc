= 무한 확장 볼륨 창과 대화 상자에 대한 설명입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
해당 상태/스토리지 가상 시스템 세부 정보 페이지에서 무한 확장 볼륨을 사용하여 SVM을 모니터링할 수 있습니다. 규칙 만들기 대화 상자에서 규칙과 데이터 정책을 관리할 수 있습니다. 스토리지 클래스 임계값 편집 대화 상자에서 스토리지 클래스 임계값을 수정할 수도 있습니다.
