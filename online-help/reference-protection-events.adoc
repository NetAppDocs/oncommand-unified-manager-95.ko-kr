= 보호 이벤트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
보호 이벤트는 작업이 실패했거나 중단되었는지 여부를 알려 사용자가 문제를 모니터링할 수 있도록 합니다. 이벤트는 영향 영역별로 그룹화되며 이벤트 및 트랩 이름, 영향 수준, 소스 유형 및 심각도를 포함합니다.
