= 규칙 템플릿의 정의
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
규칙 템플릿은 데이터 정책에서 규칙을 만드는 데 사용할 수 있는 미리 정의된 템플릿입니다. 규칙 템플릿을 사용하면 소유자, 파일 형식 및 디렉터리 경로의 세 가지 범주를 기반으로 규칙을 만들 수 있습니다.



== 파일 형식에 대한 규칙 템플릿의 예

규칙 템플릿 ""지정된 확장자를 가진 모든 파일을 적절한 저장소 클래스에 배치""는 무한 확장 볼륨에 쓴 모든 .mp3 파일을 지정한 저장소 클래스에 배치합니다.
