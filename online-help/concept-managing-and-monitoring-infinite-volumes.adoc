= 무한 확장 볼륨 관리 및 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
무한 확장 볼륨을 사용하여 SVM(스토리지 가상 머신)의 용량과 가용성을 모니터링할 수 있습니다. 규칙 및 데이터 정책을 생성하여 무한 확장 볼륨을 통해 스토리지 가상 시스템(SVM)의 콘텐츠 배치를 관리할 수 있습니다.
