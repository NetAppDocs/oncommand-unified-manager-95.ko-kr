= 네임스페이스 미러 구성요소란 무엇입니까
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
네임스페이스 미러 구성요소란 무한 확장 볼륨에 있는 네임스페이스 구성요소의 클러스터 내 데이터 보호 미러 복사본입니다. 네임스페이스 미러 구성요소에서는 두 가지 역할을 수행합니다. 즉, 네임스페이스 구성요소에 대한 데이터 보호를 제공하며 무한 확장 볼륨의 증분 테이프 백업에 SnapDiff를 지원합니다.
