= 엔클로저 이벤트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
엔클로저 이벤트는 데이터 센터에서 디스크 쉘프 엔클로저의 상태에 대한 정보를 제공하여 잠재적인 문제를 모니터링할 수 있습니다. 이벤트는 영향 영역별로 그룹화되며 이벤트 및 트랩 이름, 영향 수준, 소스 유형 및 심각도를 포함합니다.
