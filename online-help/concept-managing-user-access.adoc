= 사용자 액세스 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
역할을 생성하고 기능을 할당하여 선택한 클러스터 개체에 대한 사용자 액세스를 제어할 수 있습니다. 클러스터 내에서 선택한 개체에 액세스하는 데 필요한 기능이 있는 사용자를 식별할 수 있습니다. 이러한 사용자만 클러스터 객체를 관리할 수 있는 액세스 권한을 제공합니다.
