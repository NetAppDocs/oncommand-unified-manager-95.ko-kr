= 볼륨 전송 속도(내역) 보고서 사용자 지정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
볼륨 전송 속도(내역) 보고서를 사용자 정의하여 볼륨의 총 전송 크기에 따라 볼륨 전송 세부 정보를 볼 수 있습니다. 또한 특정 요일의 볼륨 전송을 볼 수도 있습니다.
