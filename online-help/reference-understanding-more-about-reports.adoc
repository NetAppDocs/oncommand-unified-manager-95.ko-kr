= 보고서에 대한 자세한 이해
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
옵션을 사용하여 보고서를 실행, 삭제, 내보내기 및 가져올 수 있습니다. 사용자 지정 보고서를 만들고 사용자 지정 보고서를 저장할 수도 있습니다. 필터링, 정렬, 그룹화, 서식 지정 등의 추가 작업을 수행할 수 있습니다.
