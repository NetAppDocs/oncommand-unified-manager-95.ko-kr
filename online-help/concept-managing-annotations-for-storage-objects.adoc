= 스토리지 객체에 대한 주석 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager에서 스토리지 객체에 주석을 다는 주석을 생성할 수 있습니다. 주석을 사용하면 중요한 리소스를 쉽게 식별하고 적절한 조치를 취할 수 있습니다. 예를 들어 그룹에 중요한 리소스를 추가하고 그룹 작업을 할당하거나 주석이 추가된 리소스 보고서를 만들 수 있습니다.
