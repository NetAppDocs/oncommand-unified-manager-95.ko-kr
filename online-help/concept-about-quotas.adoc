= 할당량 정보
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
할당량을 통해 사용자, 그룹 또는 qtree가 사용하는 파일 수와 디스크 공간을 제한하거나 추적할 수 있습니다. 를 사용하여 할당량을 지정합니다 `/etc/quotas` 파일. 할당량은 특정 볼륨 또는 qtree에 적용됩니다.
