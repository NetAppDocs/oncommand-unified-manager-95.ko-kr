= 사용자 액세스 창 및 대화 상자에 대한 설명입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
RBAC 설정에 따라 관리/사용자 페이지에서 사용자를 추가하고 해당 사용자에게 적절한 역할을 할당하여 클러스터에 액세스하고 모니터링할 수 있습니다.
