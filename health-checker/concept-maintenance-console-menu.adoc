= 유지보수 콘솔 메뉴
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
유지보수 콘솔은 Unified Manager 서버의 특수 기능 및 구성 설정을 유지 관리 및 관리할 수 있는 다양한 메뉴로 구성되어 있습니다.

Unified Manager를 설치한 운영 체제에 따라 유지보수 콘솔은 다음 메뉴로 구성됩니다.

* Unified Manager 업그레이드(VMware만 해당)
* 네트워크 구성(VMware만 해당)
* 시스템 구성(VMware만 해당)
* 지원/진단
* 서버 인증서를 재설정합니다
* 외부 데이터 공급자
* 성능 폴링 간격 구성
